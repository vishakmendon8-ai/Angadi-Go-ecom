A visually stunning, high-performance eCommerce application for a next-generation tech brand. Focused on robotics, drones, and AI devices with immersive 3D visualizations.

## ✨ Features

- **Cinematic 3D Hero**: Interactive Three.js scene for immediate visual impact.
- **Interactive 3D Product Viewer**: Rotate and inspect high-end hardware in real-time.
- **Glassmorphism Design**: Modern, premium UI/UX with neon aesthetics and deep shadows.
- **Full eCommerce Lifecycle**: Product discovery, shopping cart, and user dashboard.
- **Secure Authentication**: Firebase-powered Google Sign-In and Email/Password flows.
- **Database Initialization**: One-click sample data loading from the dashboard.

## 🚀 Quick Start

### 1. Prerequisites
- Node.js (v18 or later)
- npm or yarn

### 2. Installation
```powershell
# Clone the repository (or copy files)
npm install
```

### 3. Environment Setup
Create a `.env.local` file in the root directory (one has been provided for you) with your Firebase credentials.

### 4. Run Development Server
```powershell
npm run dev
```

## 🛠️ Tech Stack
- **Frontend**: React 19, Vite
- **Styling**: Tailwind CSS (Custom components plugin)
- **Animations**: Framer Motion
- **3D Engine**: Three.js, React Three Fiber, Drei
- **Backend/DB**: Firebase Authentication, Cloud Firestore

## 📂 Project Structure
- `src/components`: Reusable UI elements (Navbar, Hero, ProductCard).
- `src/context`: State management for Auth and Cart.
- `src/pages`: Full page views (Home, Products, ProductDetail).
- `src/lib`: Firebase and other service configurations.
- `src/index.css`: Global base styles (Cleaned of IDE warnings).

## 🔐 Security
- Environment variables for sensitive configuration.
- Custom Firestore security rules (`firestore.rules`).
